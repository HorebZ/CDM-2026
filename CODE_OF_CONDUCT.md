# Code of Conduct

This project follows a simple principle: **be respectful**.

In issues, pull requests, and any other discussion space related to this repository, everyone is expected to:

- Be kind, patient and welcoming, regardless of skill level or background.
- Assume good intent and give constructive feedback.
- Avoid personal attacks, harassment, discriminatory language or off-topic drama.

Unacceptable behaviour may lead to comments being removed, and, in repeated cases, to the author being blocked from interacting with the repository.

If you witness or experience behaviour that violates this code of conduct, please open a private contact with the maintainer via GitHub.

This document is inspired by the spirit of the [Contributor Covenant](https://www.contributor-covenant.org/) without adopting it formally, to keep things lightweight and adapted to the scale of the project.
